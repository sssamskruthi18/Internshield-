# Fake Internship Offer Detection

## Project Overview
This project detects whether an internship/job offer is legitimate or fraudulent using Machine Learning.

The system analyzes job descriptions and predicts whether the internship offer is real or fake.

## Technologies Used
- Python
- Google Colab
- Machine Learning
- Naïve Bayes Algorithm
- TF-IDF Vectorization

## Dataset
Dataset used: Fake Job Postings Dataset from Kaggle

## Methodology
1. Data preprocessing
2. Text cleaning
3. TF-IDF vectorization
4. Naïve Bayes model training
5. Dataset balancing using Random Oversampling
6. Prediction of fraudulent or legitimate internship offers

## Results
- Accuracy: ~95%
- Fraud detection recall: ~96%

## Sample Output
- Fraudulent Internship Offer
- Legitimate Internship Offer

## Conclusion
The model successfully detects fake internship offers with high accuracy and can help students identify fraudulent opportunities.
